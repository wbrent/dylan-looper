Performer MIDI to GUI mappings

MIDI Sustain Pedal: open mic gate

CC 50: set target to looping buffer A
CC 51: set target to looping buffer B
CC 52: set target to live input

CC 102: looping buffer A gain
CC 103: looping buffer B gain
CC 104: live input gain
CC 105: master gain

CC 110: looping buffer A decay rate
CC 111: looping buffer B decay rate

CC 11: looping buffer A clear buffer
CC 15: looping buffer B clear buffer
